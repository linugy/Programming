1.绝对路径表示：/etc/a
2.当前目录：./
3.上层目录：../
4.主目录：~/
5.切换目录：cd
6.查看当前路径：pwd
7.清屏：clear
8.帮助指令：man
9.列出文件及目录：ls
10.建立软链接：ln -s slink source
11.建立硬链接：ln link source
12.创建目录：mkdir
13.创建文件：touch
14.复制文件：cp（连同文件夹一起复制cp -r）
15.修改文件权限：chmod
16.终端在哪个路径：/dev/tty
17.移动文件：mv
18.改名：mv
19.删除文件：rm（连同目录和文件一起删除rm -r）
20.删除空文件夹：rmdir
21.通配符：？替代单个字符；*替代多个字符
22.终止进程：kill
23.查看使用过的命令列表：history
